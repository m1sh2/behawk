Configuration

Joomla (2.5, 3+) and Virtuemart 2

1. Download pack from here.

2. Unpack and upload file "behawk.php" to your default Joomla template (example: /templates/your_template/behawk.php).

3. Setup application in Settings area.

 

Settings fields

Shop name

How do you want to name your eCommerce website.

 

Shop default file name

Name of the file which to upload to the template folder (example: /templates/your_template/behawk.php). Rename this file for security reasons.

 

The protocol

http://, https://, http://www., https://www.

 

Shop url

Link to your website without protocol and backslash "/" (example: https://yourwebsite.com/, Shop url is: yourwebsite.com).

 

CMS type

Select Joomla+Virtuemart.

Opencart

2. Unpack and upload file "behawk.php" to your product folder (example: /catalog/controller/product/behawk.php).

3. Setup application in Settings area.

 

Settings fields

Shop name

How do you want to name your eCommerce website.

 

Shop default file name

Name of the file which to upload to the template folder (example: /catalog/controller/product/behawk.php). Rename this file for security reasons. Also you'll need to rename class in behawk.php file.

Example

Current file name: behawk.php

Current class name: ControllerProductBehawk

New filen ame: behawknew.php

New class name: ControllerProductBehawknew

http://, https://, http://www., https://www.
